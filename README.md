# React Login Screen

Projeto de tela de login feito com React.

## Tecnologias usadas

- React
- Vite
- CSS

## Funcionalidades

- Header com navegação
- Formulário de login
- Layout com imagem de fundo
- Efeito glassmorphism no formulário

## Como rodar o projeto

1. Clone o repositório

2. Instale as dependências

3. Rode o servidor
